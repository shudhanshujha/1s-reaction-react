# 1 Second Reaction

A mobile-friendly reflex timing game built with React and Vite.

## Game Features

- **Reflex Timing**: Stop the bar in the perfect zone to score.
- **Scaling Difficulty**: Bar speed increases and the zone shrinks with each hit.
- **Coin System**: Earn coins based on your score.
- **Skins Shop**: Buy and equip different bar styles (Neon, Fire, Electric, Gold).
- **Simulated Ads**: Watch ads to earn extra coins.
- **Local Progress**: Saves your best score, coins, and purchased skins.
